## Available GPTs

### 1. PenTestGPT (Web Apps & APIs)
Professional pentesting copilot for web applications and APIs with test case generation and professional reporting.

**Setup:** [Instructions](./penetration-testing/instructions.txt) | **Learn:** [Usage Guide](./penetration-testing/usage-guide.md)

### 2. CodeSecGPT - Secure Code Reviewer
Reviews application code for security vulnerabilities and provides secure rewrites.

**Setup:** [Instructions](./code-security/instructions.txt) | **Learn:** [Usage Guide](./code-security/usage-guide.md)

### 3. JIRATicketGPT - Security JIRA Ticket Generator
Converts vulnerability findings into structured JIRA tickets automatically.

**Setup:** [Instructions](./security-workflow/jira/instructions.txt) | **Learn:** [Usage Guide](./security-workflow/jira/usage-guide.md)

### 4. ThreatModelGPT - Threat Modeling Assistant
Creates comprehensive threat models using STRIDE + DREAD scoring with visual threat diagrams.

**Setup:** [Instructions](./threat-modeling/instructions.txt) | **Learn:** [Usage Guide](./threat-modeling/usage-guide.md)

### 5. ComplianceGPT - Compliance Assistant
Maps controls to frameworks (SOC 2, ISO 27001, PCI-DSS, HIPAA, GDPR), identifies gaps, and generates audit-ready documentation.

**Setup:** [Instructions](./compliance/instructions.txt) | **Learn:** [Usage Guide](./compliance/usage-guide.md)

## How to Use

**In ChatGPT:**
1. Go to https://chat.openai.com
2. Click your profile → "My GPTs" → "Create a GPT"
3. Click "Configure"
4. Open the GPT's `instructions.txt` file (links above)
5. Copy everything and paste into the "Instructions" field
6. Add the GPT name (e.g., `Security JIRA Ticket Generator` or `Secure Code Reviewer`)
7. Click "Save"

**In Claude:**
1. Create a new Project
2. Copy everything from the GPT's `instructions.txt` file
3. Paste into "Project knowledge"
